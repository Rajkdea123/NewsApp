## Features:

- **Hindi News Articles**: Access a wide range of news articles in Hindi sourced from a reliable API.
- **Categories**: Explore news articles categorized into different topics such as politics, sports, entertainment, technology, and more.
- **Search Functionality**: Search for specific news articles using keywords or topics of interest.
- **Detailed Article View**: Read full news articles with a clean and user-friendly interface.
- **Save for Later**: Bookmark favorite articles to read later at any time.
- **Share News**: Share interesting news articles with friends and family via various social media platforms.
- **Responsive Design**: Enjoy a responsive UI that adapts to different screen sizes and orientations, ensuring a consistent experience across devices.

## Technologies Used:

- **Flutter**: A cross-platform UI toolkit for building natively compiled applications for mobile, web, and desktop from a single codebase.
- **GetX**: A lightweight, yet powerful state management solution for Flutter that simplifies development and enhances performance.
- **Hindi News API**: Utilize a reliable Hindi news API to fetch and display up-to-date news articles in the app.

## Getting Started:

To get started with the project, follow these steps:

1. Clone the repository to your local machine.
2. Install Flutter and Dart on your development environment.
3. Run `flutter pub get` to install dependencies.
4. Obtain an API key from the Hindi news API provider and replace the placeholder in the code with your API key.
5. Run the app on your preferred device or simulator.
